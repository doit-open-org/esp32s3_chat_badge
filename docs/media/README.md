# Demo videos

This directory contains narrated demonstrations of the ESP32-S3 Chat Badge working with the “四博小助手” WeChat Mini Program.

- `esp32s3_chat_badge_demo_zh.mp4`: Chinese narration
- `esp32s3_chat_badge_demo_en.mp4`: English narration
- `narration_zh.txt`: Chinese narration script
- `narration_en.txt`: English narration script

The two source recordings are combined into one walkthrough. The long image-upload wait is accelerated, and the idle delay between the wake phrase and the device response is removed.
